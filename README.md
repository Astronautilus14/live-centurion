# Live centurion

## The game: Centurion
Centurion is a drinking game where you listen to the [centruion mix](https://soundcloud.com/gebroedersscooter/totale-escalatie-centurion) by DJ duo the [Gebroeders Scooter)](https://soundcloud.com/gebroedersscooter). Trougout the mix there are 100 airhorn, about 1 minute seperated. Everytime you hear an airhorn, you have to drink. This app is made for DJs to keep track of the airhorns played while mixing there own version of this game live.

## Feaatures
 - Airhorn count
 - Time since last airhorn
 - Pause button
 - Persistant state trough local storage (for drunk DJs accidantially reloading the website)
 - Used audio context to play airhorn without latency (idk AI made that part, but it work👍)
