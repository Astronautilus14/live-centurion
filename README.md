# Live centurion

Centurion is a drinking game where you listen to the [centruion mix](https://soundcloud.com/gebroedersscooter/totale-escalatie-centurion) by DJ duo the [Gebroeders Scooter](https://soundcloud.com/gebroedersscooter). Throughout the mix 100 airhorns are played, about 1 every minute. Everytime you hear an airhorn, you have to drink. This app is made for DJs to keep track of the airhorns played while mixing their own version of this game live.

## Features
 - Airhorn count + timer
 - Second screen to be put on a big screen (e.g. TV) for the audiance with ✨special✨ effects using the Broadcast Channel API
 - Persistant state trough local storage (for drunk DJs accidantially reloading the website)
 - Used audio context to play airhorn without latency (idk AI made that part, but it works👍)

# Development
You need to have node.js installed

Install dependencies 
```bash
npm i
```

Run development server
```bash
npm run dev
```

And that's all
